# ir

Install the **Extension**: In VS Code, go to the **Extensions** view (Ctrl+Shift+X) and search for **Live Share** (by Microsoft) and install it.

Sign in: A "Live Share" icon will appear in your activity bar. Click it and sign in with your GitHub or Microsoft account.

Open Project: Open the GitHub repository folder you want to share in VS Code.

Start Session: Click the Live Share icon in the activity bar and select Start collaboration session. A link will be automatically copied to your clipboard.

Share Link: Send this invitation link to your teammates via chat, email, or any other platform.
